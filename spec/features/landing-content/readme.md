# Landing page — conteúdo editorial

Especificação canônica dos textos visíveis da landing page (`app/landing-page.tsx`).

## Princípios

- Tom acolhedor, comercial na medida, compatível com psicologia clínica.
- Manter estilo e extensão próximos ao copy existente; alterar somente o que estiver explicitamente listado em cada ciclo.
- Usar `&` nos títulos de temas (ex.: `Ansiedade & Angústia`).
